Lumo is a secure communication and productivity assistant designed with privacy in mind.

Key Features:
- End-to-end encrypted messaging
- Secure file sharing
- Privacy-focused AI assistance
- Open-source codebase for transparency

This app respects user sovereignty and data ownership. All data is processed locally where possible, and no personal information is shared without explicit consent.

For more information, visit our official website or check the source code on GitHub.
